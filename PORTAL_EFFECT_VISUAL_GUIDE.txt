┌──────────────────────────────────────────────────────────────────────────────┐
│ PARAMETER TUNING GUIDE                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

  Problem: Effect too subtle
  Solution: ↑ Falloff (2.0 → 4.0)
           ↓ Inner Radius (0.5 → 0.3)

  Problem: Effect too sharp
  Solution: ↓ Falloff (3.0 → 1.0)
           ↑ Distance between radii

  Problem: Portal too small
  Solution: ↑ Inner Radius
           ↑ Outer Radius

  Problem: Portal too large
  Solution: ↓ Inner Radius
           ↓ Outer Radius

  Problem: Effect doesn't move with object
  Solution: Ensure shader was saved and recompiled
           Check Unity console for errors

╚══════════════════════════════════════════════════════════════════════════════╝

  Component:  GaussianSplatPortalController
  
  Parameters:
    • Enable Portal Fade  [✓/✗]      Toggle effect
    • Portal Center       [Vector3]   Object-space position
    • Inner Radius        [0-5]       No fade zone
    • Outer Radius        [0-10]      Full fade distance
    • Falloff             [0.1-5]     Fade curve sharpness
  
  Shortcuts:
    • Tight Portal:   Inner=0.2, Outer=0.5, Falloff=3.0
    • Soft Vignette:  Inner=0.8, Outer=2.0, Falloff=1.0
    • Balanced:       Inner=0.5, Outer=1.0, Falloff=2.0
  
  Tips:
    ✓ Select object to see gizmos
    ✓ Adjust in real-time
    ✓ Works with object movement
    ✓ Center is in object space (0,0,0 = center)
